The Stanford cslib package

hashmap.h

This interface defines a map abstraction that associates string keys with values. The implementation uses a hash table, which offers constant-time performance but does not support iterating through the keys in order.

void map(HashMap map, proc fn, void *data);
Iterates through the map and calls the function fn on each entry. The callback function takes the following arguments:
  • The key string
  • The associated value
  • The data pointer
The data pointer allows the client to pass state information to the function fn, if necessary. If no such information is required, this argument should be NULL.

Usage:

map(map, fn, data);
